### 4. 스터디 규칙 안내

- **사전에 정한 목요일 15:00**에 진행됩니다.

- 매주 한 명당 **1번의 강의 리뷰**를 진행합니다.  

- 강의를 듣지 않으면 의미가 없습니다. 이를 경계하기 위해 다음과 같은 규칙을 만들었습니다. 
    1. 매주 **각 강의마다 최소 1개의 질문거리**를 갖고 올 것

     >- [x] **강의 중 이해가 잘 되지 않는 부분**
